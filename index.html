📏 CSS Units: px, em, rem

Understanding how different CSS units work helps create responsive and scalable designs.

px – Absolute unit, fixed pixel size (e.g., 16px is always 16 pixels).

em – Relative to the parent element’s font size. Can compound when nested.

rem – Relative to the root (<html>) element’s font size. Consistent and does not compound.

🔍 Difference Between em and rem

em: Scales based on the parent’s font size. Nesting can multiply sizes.
rem: Always based on the root font size, regardless of nesting, making it more predictable.

Example of em compounding:

  • Body font-size: 16px
  • div font-size: 1.5em (1.5 × 16px = 24px)
  • p inside div font-size: 2em (2 × 24px = 48px)

Example of rem consistent sizing:

  • Root font-size: 16px
  • div font-size: 1.5rem (1.5 × 16px = 24px)
  • p inside div font-size: 2rem (2 × 16px = 32px)
